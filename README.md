# 💬 ChatBridge

A scalable real-time chat application built with Python and JavaScript, enabling seamless communication using WebSockets, secure authentication, and efficient message delivery.

---

## 🚀 Features

- ⚡ Real-time messaging using WebSockets  
- 🔐 Secure authentication system  
- 🌐 Python backend + JavaScript frontend  
- 📡 Low-latency communication  
- 🧠 Scalable architecture  

---

## 🛠️ Tech Stack

- Backend: Python (Flask / FastAPI)  
- Frontend: JavaScript  
- Communication: WebSockets  
- Others: REST APIs, JSON
